## How to start

* This project requires _Node 6_ and _NPM 3.10 or greater_.

1. Point `DATABASE_URL` in `server/constants.js` to your MongoDB.

```shell
# In project root
npm i
npm i -g gulp http-server typescript
gulp
npm run start
```
